# Back-End

- Node
- Express

# App

- React-native
- Axios
- Redux
- Redux-saga
- React-navigation
- Styled-components

# Run Back-End

- clone repository

```
  - cd node-app-docker
  - docker-compose up -d
  - cd ..
```

# Run App

```
  - cd client
  - yarn or npm install
  - yarn ios or yarn android
```
